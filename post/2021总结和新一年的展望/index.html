2021总结和新一年的展望

时间过得很快，元旦仿佛还在昨天，而现在已经是1月下旬了。伟人说过一万年太久，只争朝夕，深以为然。然而除了埋头赶路之外，还得时常停下脚步审视自己走过的路，况且2021年对我而言是巨变的一年，实在值得回顾一番

生活

结婚

关于我和rua结婚这件事情真是说来话长。因为疫情的关系，原定于2020年年初的婚礼被硬生生拖了将近两年，好在去年十一我们终于结婚了！进入婚姻关系除了喜悦之外更多的是责任。余生很长，无论前路是阳关大道还是荆棘小径，都愿与你相互扶携，共同欣赏绚丽的人生风景。

定居

说到定居要特别感谢我的领导们。听说我在杭州买了房之后，主动询问我是否愿意从北京trans到杭州。于是年中结束了北漂生活，来到了杭州与rua团聚。从此下班之后不用困在不到10平米的小房子里了，真TM爽！

工作

主业

参加工作也有7年了，在此过程中也走过一段不长不短的弯路，有幸进入到现在这家公司，从事着自己喜欢的技术方向，每天享受着持续成长带来的成就感，也从未像今天这样有如此明确的目标感和求知欲。

今年主要干了几件事情。

  • ClickHouse运维管理平台。目前已经在内部用了半年多了，之前和团队小伙伴商量把这套系统开源出去，奈何其他事情太多，搁置到现在，希望今年能立项启动
  • ClickHouse Hive Table Engine。实现了ClickHouse对Hive数仓的查询。
  • 持续贡献ClickHouse社区：贡献了供17个pr, 包括Hive Table Engine(#31104), MergeTree启动加速(#32928)； 提了12个issue。

PS: 年会上获得了优秀员工，算是一直以来努力工作的一个肯定吧。我虽不是大佬，但一直在向大佬们学习的路上。

副业

2021年开启了副业的探索，目前尝试了技术写作和接私活这两条路

写作

微信公众号

今年和同事创建了两个微信公众号

  • ClickHouse峡谷
  • 后端技术小屋

总共写了79篇原创文章，一共700多关注，离年初1k关注的目标还是有点距离。其中6篇文章投稿

知乎

平时写文章的主阵地是微信公众号，当然也会顺手转到知乎上。知乎上的阅读量一直不温不火，直到年尾两篇《optimize c++ software》的读书笔记突然迎来了意料之外的关注和收藏，现在关注量已经达到了355。整体感觉知乎对于小号在流量的分配上还是比较友好的，只要有干货就不愁没人关注。

其他

我的文章一开始也会转到掘金、简书、博客园上。然而掘金上充斥了各种标题党和八股文、简书审核太严格、博客园流量太低，总之感觉相当鸡肋，所以后来干脆放弃了。

接活

2021年运气不错，从同学那儿接到了几单私活儿，挣了点意外之财，也不多才几k。希望今年也能有此好运。
