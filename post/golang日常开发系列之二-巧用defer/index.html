golang日常开发系列之二--巧用defer

2021-07-18
GO

我们都知道，在golang中，有一种语言特性跟C++中的RAII特别相似，那就是defer。例如以下两段代码便可实现相同的效果

#include <iostream>

class Defer {
    public:
        Defer() { }

        ~Defer() {
            std::cout << "defer" << std::endl;
        }

};

int main() {
    Defer d;
    return 0
}

import "fmt"

func main() {
	defer func() {
		fmt.Println("defer")
	}()
}

上述两段代码都会在main函数退出时，打印出字符串"defer”
