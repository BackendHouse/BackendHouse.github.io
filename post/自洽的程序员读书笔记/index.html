沟通

沟通分享的时机 -> 下午茶 散步 旅行
沟通分享的方法 -> 尊重 平等 不抱怨 及时反馈

维系亲密关系

接纳对方本来的样子

  • 包容不完美
  • 给于成长的空间
  • 不频繁批评指责

接纳对方的话术

  • 你喜欢就好，我支持你
  • 这是你的选择，我理解
  • 需要帮忙随时说
  • 你有自己的想法很好
  • 也许我们可以保留各自意见
  • 我试着理解你的观点
  • 我们想法不同，这很正常

接纳的边界

  • 不是放纵伤害
  • 不是忽视原则问题
  • 不是牺牲自我价值

情绪

如何平复情绪

  • 深呼吸 冥想
  • 运动 户外
  • 写日记
  • 找朋友倾诉

工作

同事太优秀怎么办

合作和竞争

  • 请客吃饭，请教技术问题
  • 在同一个项目中交流学习
  • 一次学会他们的一个细分技能，持续进步

竞争观

  • 永远不要因为别人太优秀而内耗，最重要的是自身的成长
  • 人生不是竞赛，而是一场自己的旅程

职场人际关系

如何打造职场人际关系

  • 工作中展现专业能力
  • 协作时表示理解和支持
  • 交往中保持真诚和诚信
  • 合作中创造共赢价值

1on1谈什么

  • 工作进展：问题，解决，收获，需要的支持
  • 个人成长：在学什么，瓶颈，困惑，未来规划
  • 团队：氛围，协作，流程，团队缺什么

工作中的第一性原理

经过多年摸爬滚打，很多人会达到一个更通透的状态:
• 不再焦虑要不要转管理(反正都是坑)
• 不再纠结要不要进大厂(大厂也裁员)
• 找到了自己的节奏(摸鱼和卷，都是人生的一部分)
• 建立了自己的判断标准(老板开心不是最重要的，自己开心才是)

第一性原理重新思考工作

  • 价值交换：时间 + 技能 + 创意 + 体力 <-> 工资 + 经验 + 人脉 + 成就感
  • 个人成长：技能树 + 认知 + 社交
  • 工作只是人生的一部分

如何应对职场倦怠和失落

1. 厌倦型的治疗方案

第一步：找回兴趣的源头

  • 回忆当初为什么选择编程
  • 列出曾经让你兴奋的技术点
  • 想想最有成就感的项目

第二步：重新培养兴趣

  • 尝试一个自己感兴趣的side project
  • 研究一下新技术或者开源项目
  • 和志同道合的同事一起做点东西

第三步：转化兴趣

  • 把枯燥的工作游戏化
  • 在日常工作中设立小目标
  • 给自己设定技术挑战

2. 失落型的治疗方案

第一步：重新定义价值

  • 不只看外在回报
  • 关注个人成长
  • 寻找工作的其他意义

第二步：创造价值

  • 主动承担重要任务
  • 解决团队痛点问题
  • 提升工作的可见度

第三步：寻找平台

  • 选择更适合的团队
  • 找到欣赏自己的领导
  • 寻找价值观一致的环境
